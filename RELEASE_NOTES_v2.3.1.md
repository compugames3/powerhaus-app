   - Máximo 80 palabras por respuesta
   - Idioma: Español
   - Actualizaciones automáticas

3. **Gestión de Conversaciones:**
   - **Minimizar (-)** = Guarda chat en sessionStorage
   - **Cerrar (×)** = Borra conversación completamente

## 🚀 Cómo Usar

### Instalación (Setup)
1. Descargar: `PowerHaus Setup 2.3.1.exe`
2. Ejecutar el instalador
3. Seguir el asistente de instalación
4. Iniciar PowerHaus desde el menú Inicio

### Ejecución Portable
1. Descargar: `PowerHaus 2.3.1.exe`
2. Ejecutar directamente (sin instalación)
3. No requiere configuración adicional

### Uso del Chatbot IA
1. Abre PowerHaus en cualquier página (excepto login/terminales)
2. **Click en botón azul** (esquina inferior derecha) - Ícono de chat
3. **Escribe tu pregunta** o usa un comando
4. **Presiona Enter** o click en botón enviar
5. **Minimiza para guardar** o **cierra para borrar**
